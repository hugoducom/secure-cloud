# secure-cloud

How to install and execute the project locally :

```sh
git clone https://github.com/hugoducom/secure-cloud.git
cd secure-cloud
# Maybe create a venv
pip install -r .\requirements.txt
python .\main.py
```

It should output the welcoming CLI :

```
Welcome to the secure cloud client!

        1. Login
        2. Register
        3. Exit

Enter your choice:
```
